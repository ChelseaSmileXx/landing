## Описание

Это учебный проект — резиновая адаптивная вёрстка страницы-презентации объекта «Фрунзенская набережная» для сайта компании «Sminex-Интеко» на основе дизайна из Figma.

## Функционал

- «Резиновая» адаптивная вёрстка для корректного отображения на любой ширине экрана (от 360px до 4000px)
- Адаптация под все разрешения, указанные в макете
- Ретинизация растровых изображений для экранов с плотностью пикселей 2x
- Стилизация с помощью препроцессора Sass (SCSS)
- Семейства шрифтов и цвета вынесены в SCSS-переменные
- Все состояния интерактивных элементов из UI-kit (hover, focus, active)
- Подвал прижат к низу экрана при недостаточной высоте содержимого
- Сетки реализованы с помощью Flexbox и Grid
- Вёрстка выполнена по методологии БЭМ с соблюдением семантики
- Векторные изображения собраны в SVG-спрайт
- Формы с корректными атрибутами (id, name, value, required)
- У интерактивных элементов есть визуальный отклик при фокусе с клавиатуры
- У спозиционированных графических элементов отключена кликабельность (pointer-events: none)
- Вёрстка проверена на переполнение контентом и валидность (HTML, БЭМ)

## Технологии

- HTML5
- CSS3 / SCSS
- Flexbox / Grid
- БЭМ
- Figma
- Live Sass Compiler

## Инструкция по запуску

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/ChelseaSmileXx/landing.git
   ```

2. Перейдите в папку проекта:
   ```bash
   cd UI
   ```

3. Откройте файл index.html в браузере (или используйте Live Server в VS Code)
